## 컴퓨터 하드웨어의 구성

중앙 처리 장치
: Central Processing Unit, <mark>CPU</mark>. 계산을 수행하고 명령어를 <u>처리</u>한다

기억 장치 
: Memory / computer data storage. 데이터와 명령어를 <u>저장</u>한다

입출력 정치
: Input/Output device, I/O장치. 데이터를 입력 받고 출력한다

➕ **시스템 버스(system bus)** 데이터를 주고 받는 <u>통로</u> - *눈에는 안 보임*
